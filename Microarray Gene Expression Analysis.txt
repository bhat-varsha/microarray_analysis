Microarray Gene Expression Analysis Using Affymetrix Data in R
Introduction

Gene expression analysis is an important part of bioinformatics used to understand how genes behave under different biological conditions such as disease vs healthy state, treated vs untreated samples, or cancer vs normal tissues.

In this workflow, Affymetrix microarray raw data from GEO was analyzed using R and Bioconductor packages. The complete pipeline included:

downloading GEO data
preprocessing raw CEL files
normalization
differential gene expression analysis
visualization using volcano plots and heatmaps
1. What is Affymetrix Data?

Affymetrix is a microarray technology used for measuring gene expression levels of thousands of genes simultaneously.

The technology uses:

DNA probes fixed on a chip
fluorescently labeled RNA/cDNA samples
scanner-generated intensity values

Each spot/probe on the microarray chip corresponds to a gene or transcript.

The output intensity values indicate:

how strongly a gene is expressed
whether genes are upregulated or downregulated
2. What is Agilent Data?

Agilent Technologies also provides microarray-based gene expression platforms.

Similar to Affymetrix:

RNA samples hybridize to probes
fluorescence intensity is measured
expression values are generated

Difference:

Affymetrix generally uses short oligonucleotide probes
Agilent generally uses longer probes

Both technologies are widely used in:

cancer studies
biomarker discovery
differential gene expression analysis

6. What is a CEL File?

CEL files are the raw output files generated by Affymetrix microarray scanners.

They contain:

raw probe intensity values
fluorescence measurements
probe-level data

CEL files are considered:

unprocessed microarray data
starting point of Affymetrix analysis

These files are later:

normalized
background corrected
summarized into gene expression values
